# Support

Use GitHub Discussions only if the repository enables them in the future. For now:

- Open a bug report for reproducible plugin behavior.
- Open a feature request for scoped product changes.
- Use GitHub private vulnerability reporting for security issues.
- Use the OpenClaw documentation and community support channels for general OpenClaw installation, model, provider, or Telegram configuration questions.

Include OpenClaw, plugin, Node.js, and operating-system versions. Redact tokens, chat identifiers, private message text, and personal data.
